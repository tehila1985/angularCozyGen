# CozyGen - פלטפורמת עיצוב פנים חכמה

## 🏠 אודות CozyGen

CozyGen היא פלטפורמת מסחר אלקטרוני חדשנית המשלבת מכירת רהיטים מודרנית עם בינה מלאכותית מתקדמת כדי ליצור חוויית עיצוב פנים מושלמת. הפלטפורמה שלנו עוזרת ללקוחות למצוא את הרהיטים האידיאליים לביתם באמצעות המלצות חכמות ופתרונות עיצוב מותאמים אישית.

**סלוגן:** "פשוט נעים · עיצוב חכם"

## ✨ תכונות עיקריות

### 🤖 עוזר עיצוב מבוסס בינה מלאכותית
- **ניתוח תמונות חכם**: העלו תמונה של החדר שלכם וקבלו המלצות מותאמות אישית לרהיטים
- **צ'אטבוט חכם**: שירות לקוחות 24/7 דובר עברית המופעל על ידי בינה מלאכותית מתקדמת
- **התאמת סגנון**: הבינה המלאכותית מזהה אוטומטית את סגנון החדר שלכם ומציעה מוצרים תואמים
- **תיאום צבעים**: התאמת צבעים חכמה על בסיס הפלטה הקיימת בחדר

### 🛍️ פלטפורמת מסחר אלקטרוני
- **קטלוג מוצרים**: אוסף רהיטים מקיף עם תיאורים מפורטים
- **חיפוש וסינון מתקדם**: סינון לפי קטגוריה, סגנון, טווח מחירים ועוד
- **עגלת קניות**: ניהול עגלה מלא עם בקרת כמויות
- **תשלום מאובטח**: אפשרויות תשלום מרובות כולל אינטגרציה עם PayPal
- **היסטוריית הזמנות**: מעקב אחר כל הרכישות וסטטוס ההזמנות

### 🎨 קטגוריות עיצוב וסגנונות
**קטגוריות רהיטים:**
- רהיטי סלון
- חדרי שינה
- חדר אוכל
- רהיטי משרד
- פתרונות אחסון

**סגנונות עיצוב:**
- **מודרני**: קווים נקיים, צבעים ניטרליים, מינימליזם
- **סקנדינבי**: עץ בהיר, פונקציונליות, פשטות
- **קלאסי**: עץ כהה, פרטים מעוטרים, אלגנטיות
- **בוהו**: צבעים חמים, טקסטילים, תערובת אקלקטית

### 👤 ניהול משתמשים
- **הרשמה והתחברות**: מערכת אימות מאובטחת
- **פרופילים אישיים**: ניהול מידע אישי והעדפות
- **מעקב הזמנות**: צפייה בהיסטוריית רכישות מלאה
- **פאנל ניהול**: ניהול מערכת מלא למנהלים

### 📱 עיצוב רספונסיבי
- **Mobile-First**: מותאם לכל גדלי המכשירים
- **UI/UX מודרני**: ממשק נקי ואינטואיטיבי עם אנימציות חלקות
- **תמיכה RTL**: תמיכה מלאה בעברית עם פריסה מימין לשמאל
- **נגישות**: מעוצב עם שיטות עבודה מומלצות לנגישות

## 🚀 מחסנית טכנולוגיות

### Frontend
- **Angular 18**: פריימוורק TypeScript מודרני
- **PrimeNG**: ספריית רכיבי UI מקצועית
- **RxJS**: תכנות ריאקטיבי לטיפול בנתונים
- **CSS3**: עיצוב מודרני עם אנימציות ומעברים

### אינטגרציית בינה מלאכותית
- **Groq API**: מודל שפה מתקדם לפונקציונליות הצ'אטבוט
- **LLaMA 3.3**: בינה מלאכותית מתקדמת לעיבוד שפה טבעית
- **Computer Vision**: ניתוח תמונות לזיהוי סגנון חדרים
- **Machine Learning**: אלגוריתמי המלצות מוצרים חכמים

## 🛠️ התקנה והגדרה

### דרישות מוקדמות
- Node.js (גרסה 18 ומעלה)
- Angular CLI
- דפדפן אינטרנט מודרני

### שלבי התקנה
```bash
# שכפול הריפוזיטורי
git clone [repository-url]

# מעבר לתיקיית הפרויקט
cd CozyGen

# התקנת תלויות
npm install

# הפעלת שרת פיתוח
ng serve

# פתיחת דפדפן ל-http://localhost:4200
```
